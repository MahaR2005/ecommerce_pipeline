# E-commerce Data Pipeline

## Overview

This project is an ETL pipeline that extracts e-commerce data,
transforms it using Python and Pandas, and loads it into PostgreSQL.

## Tech Stack

- Python
- Pandas
- PostgreSQL
- SQLAlchemy
- Linux
- SQL

## Pipeline Flow

CSV File
   |
   ↓
Extract
   |
   ↓
Transform
   |
   ↓
Load
   |
   ↓
PostgreSQL Database

## Features

- Data extraction from CSV
- Data cleaning and transformation
- Database loading
- Error handling
- Logging

## Run Project

Install dependencies:

pip install -r requirements.txt

Run:

python load.py
